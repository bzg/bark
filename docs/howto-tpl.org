* 🐕 BARK --- Bug And Report Keeper

BARK monitors a mailbox and turns incoming emails into reports.
Follow-up emails in the same thread can update that report's state
through trigger words. Admins and maintainers manage roles and can
receive periodic digest email notifications.

For the full documentation, see the [[https://codeberg.org/bzg/bark][BARK project page]].

** Labels and triggers

| Type         | Subject labels                                  | Acked              | Owned   | Closed               |
|--------------+-------------------------------------------------+--------------------+---------+----------------------|
| bug          | =[BUG]= =[BUG version]=                             | =Approved= =Confirmed= | =Handled= | =Canceled= =Fixed=       |
| patch        | =[PATCH]= =[PATCH n/m]= =[PATCH topic n/m]=           | =Approved= =Reviewed=  | =Handled= | =Canceled= =Applied=     |
| request      | =[POLL]= =[FR]= =[TODO]=                              | =Approved=           | =Handled= | =Canceled= =Done= =Closed= |
|--------------+-------------------------------------------------+--------------------+---------+----------------------|
| announcement | =[ANN]= =[ANNOUNCEMENT]=                            |                    |         | =Canceled=             |
| release      | =[REL]= =[RELEASE]= =[REL version]= =[RELEASE version]= |                    |         | =Canceled=             |
| change       | =[CHG]= =[CHANGE]= =[CHG version]= =[CHANGE version]=   |                    |         | =Canceled=             |
